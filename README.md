## Install
- Create database blog
- Import dump_sql.sql
- Config .env_demo and rename it to .env
- Run composer update :
 ```
composer update
 ```
- Run maildev :
 ```
maildev
 ```
- Execute the project with :
 ```
php -S 127.0.0.1:8000
 ```
